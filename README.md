## How to run?

- Create the "config.json" file in the /src dir:

  ```
  {
    "token": "",
    "clientId": "",
    "guildId": ""
  }
  ```

- Install node modules:

    `npm i`

- Run the bot:

    `npm run start`

### Should look like this:

```
Starting compilation in watch mode...
Found 0 errors. Watching for file changes.
Ready!
Logged in as ...!
```

### Reloading slash commands:

`npm run reload`
